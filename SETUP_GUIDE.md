## 🔐 STEP 1: GitHub Secrets (REQUIRED)

Go to: `GitHub Repo → Settings → Secrets and variables → Actions`

Add these 6 secrets:

| Secret Name | How to Get It | What It Is |
|-------------|--------------|------------|
| `GOOGLE_SERVICES_JSON` | Firebase Console → Project Settings → Android App → Download | Full JSON content of google-services.json |
| `FIREBASE_OPTIONS_DART` | Run `flutterfire configure` locally | Full content of lib/firebase_options.dart |
| `KEYSTORE_BASE64` | `base64 -w 0 rozgarx-release.jks` | Your keystore file in base64 |
| `KEY_STORE_PASSWORD` | Your password when creating keystore | Keystore password |
| `KEY_PASSWORD` | Your key password | Key password |
| `KEY_ALIAS` | Your alias name | e.g., "rozgarx" |

### How to Create Keystore:
```bash
keytool -genkey -v \
  -keystore rozgarx-release.jks \
  -keyalg RSA -keysize 2048 \
  -validity 10000 \
  -alias rozgarx

# Then encode to base64:
base64 -w 0 rozgarx-release.jks
# Copy the output → paste as KEYSTORE_BASE64 secret
```

## 🔥 STEP 2: Firebase Setup

### 2a. Create Project
1. Go to https://console.firebase.google.com
2. Create Project → "RozgarX"

### 2b. Enable Services
In Firebase Console, enable ALL of these:
- ✅ Authentication → Anonymous + Email/Password + Google Sign-In
- ✅ Firestore Database → Start in production mode
- ✅ Cloud Messaging (FCM) → Already enabled
- ✅ Cloud Functions (REQUIRES BLAZE PLAN for external network)
- ✅ Analytics → Enable
- ✅ Crashlytics → Enable
- ✅ Remote Config → Enable

### 2c. Add Android App
1. Project Settings → Add App → Android
2. Package: `com.rozgarx.app`
3. Download `google-services.json`
4. Content goes to `GOOGLE_SERVICES_JSON` secret

### 2d. Generate firebase_options.dart
```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure (run in flutter_app directory)
cd flutter_app
flutterfire configure --project=your-firebase-project-id

# This generates lib/firebase_options.dart
# Copy its content to FIREBASE_OPTIONS_DART secret
```

### 2e. Firestore Security Rules
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    match /jobs/{jobId} {
      allow read: if true;
      allow write: if false; // Only Cloud Functions write jobs
    }
    
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
    
    match /logs/{logId} {
      allow read: if request.auth != null;
      allow write: if false;
    }
  }
}
```

### 2f. Firestore Indexes (Required for queries)
Create composite indexes:
1. Collection: `jobs` | Fields: `last_date` ASC, `created_at` DESC
2. Collection: `jobs` | Fields: `category`, `last_date` ASC
3. Collection: `jobs` | Fields: `state`, `last_date` ASC
4. Collection: `jobs` | Fields: `is_featured`, `last_date` ASC

## 🐛 TROUBLESHOOTING

| Error | Solution |
|-------|----------|
| `google-services.json not found` | Add GOOGLE_SERVICES_JSON secret |
| `firebase_options.dart not found` | Add FIREBASE_OPTIONS_DART secret |
| `Keystore not found` | Add KEYSTORE_BASE64 + KEY_* secrets |
| `Gradle build failed` | Check Java 17 is selected in workflow |
| `Functions deploy failed` | Enable Blaze plan in Firebase |
| `No jobs appearing` | Wait 3 hours for auto-fetch OR seed data |
| `FCM not working` | Verify google-services.json has messaging enabled |
| `AdMob ads not showing` | Switch `_isDebug = false` and use real IDs |
